# Post Quantum Cryptography - Learning With Errors

This repository showcases the implementation of various LWE variants (LWE, RLWE). These algorithms are used in PQC standards like [ML-KEM](https://en.wikipedia.org/wiki/ML-KEM). And while I build on the same foundations, it should be noted that I do not guarantee this code is cryptographically secure.

This project has been developed in Ruby. (Why? Because I needed to learn Ruby!)
